# Expense Tracker

A modern expense tracking app built with React, Vite, Tailwind CSS, and Chart.js.
It helps you log, visualize, and analyze your spending with dark mode, filters, and CSV export.

## Live Demo
[https://expense-tracker-react-dusky.vercel.app/]

## Features

- Add / Delete Expenses with description, amount , category, and date
- Dark Mode Toggle 
- Dashboard & Charts
- Filters
- Demo Data Loader
- CSV Export 
- Persistent State via localStorage

## Tech Stack

- React + Vite
- Tailwind CSS
- Chart.js + react-chartjs-2
- LocalStorage

## Getting Started
1. Clone the repo:
  ```bash
  git clone https://github.com/pdvladking/expense-tracker-react.git
  cd expense-tracker-react

2. Intall dependencies
  npm install

3. npm run dev

4. npm run build
